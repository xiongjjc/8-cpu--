## MOV指令
| 寻址方式         | 示例指令        |
|------------------|-----------------|
| 立即寻址         | MOV  A, 5;     |
| 寄存器寻址       | MOV  A, B;     |
| 直接寻址         | MOV  A, [5];   |
| 寄存器间接寻址   | MOV  A, [B];   |

---
